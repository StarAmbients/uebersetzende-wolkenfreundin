# Project Overview

**Übersetzende Wolkenfreundin – The Cloud Translator Friend**

Purpose: AI‑assisted, human‑verified translation pipeline to help the Friends of Bruno Gröning share reports across languages, starting with **German → Portuguese (Brazil)**.

Status: **Phase 1 complete** (environment + knowledge base preview).

Highlights:
- Cloud‑native pipeline (Cloud Run, Cloud Storage, Vertex AI Search + Gemini)
- Multi‑provider translations (DeepL, Microsoft Translator)
- Reviewer‑friendly DOCX outputs (side‑by‑side), human‑in‑the‑loop QA
